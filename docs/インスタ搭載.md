# Instagram 投稿の埋め込み

Branchサイトのトップページに、選んだInstagram投稿を埋め込み表示します。
運営がNotionで「見せたい投稿」を選べる方式です（無料・外部サービス登録不要）。

## 表示のしくみ

- トップページ下部に「Instagram」セクションが出ます
- **公開中の投稿が0件かつプロフィールURLも未設定のときは、セクション自体が非表示**になります（空欄が出ない）

## 投稿を載せる手順

1. Instagramで載せたい投稿を開き、**投稿のURLをコピー**
   - 例：`https://www.instagram.com/p/ABC123/`（リール `/reel/...` やIGTV `/tv/...` もOK）
2. 「Branch インスタ投稿DB」（`3a776a17-0aae-818e-8b00-ea3294ee042f`／「サイト・講座」内）で ＋新規
3. タイトル（管理用の覚え書き）と **投稿URL** を入力
4. **公開** にチェック → 次のサイト反映（1時間以内）で表示されます
5. 並び順は「表示順」（小さい順）で調整

載せるのをやめたいときは **公開のチェックを外す**だけです。

> URLが投稿ページでない（プロフィールURL等）の場合は、自動でスキップされ表示されません。

## フォロー導線（プロフィールへのリンク）

「Instagramでフォローする」ボタンを出すには、
`src/consts.ts` の `INSTAGRAM_URL` にプロフィールURLを設定します。

```
export const INSTAGRAM_URL = 'https://www.instagram.com/あなたのアカウント/';
```

空のあいだはフォローボタンは表示されません。設定後にコミットすれば反映されます。

## 補足

- 埋め込みはInstagram公式の埋め込み枠（iframe）を使っています。**投稿を非公開・削除**すると
  サイト上でも表示されなくなります（Instagram側の仕様）。
- サイトのビルドは不要で、Notionの公開チェックだけで出し入れできます。
