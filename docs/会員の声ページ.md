# 会員の声・実績ページ

受講生の感想と実績を掲載する「教育ページ」です。検討中の方に、学んだ人が
どう変わったかを見てもらうことで入会を後押しします。

- 公開URL: `https://mayotesoh.github.io/Branch-site/voices`
- ヘッダーの「会員の声」からアクセスできます

## 使うDB

**Branch 会員の声DB**（`3a776a17-0aae-81e8-8abb-d889d401e589`／「サイト・講座」ページ内）

| プロパティ | 内容 |
|---|---|
| 表示名 | サイトに出す名前（本名でなくてもOK。例：M.Sさん、占い師名） |
| 肩書き | 例：タロット占い師／会社員から独立 |
| 顔写真 | 任意。無い場合は記号が表示されます |
| 実績 | 目立つ枠で表示されます。例：受講3ヶ月で月商20万円 |
| 感想 | 本文。改行はそのまま反映されます |
| 受講講座 | 講座DBから選択。カードの下部にタグ表示 |
| 会員 | 会員DBと紐付け（任意・管理用） |
| **掲載許可** | **ご本人の許可が取れたらチェック** |
| **公開** | サイトに出すときにチェック |
| 表示順 | 小さい順に並びます |

> **⚠️ 掲載は「掲載許可」と「公開」の両方にチェックが入ったものだけ**が表示されます。
> 実在の方の声を扱うため、必ずご本人の同意を得てから掲載してください。
> 写真・実績・金額の掲載可否も個別に確認することをおすすめします。

## 掲載の手順

1. 「Branch 会員の声DB」で ＋新規
2. 表示名・肩書き・実績・感想を入力（受講講座も選べます）
3. ご本人の同意が取れたら **掲載許可** にチェック
4. **公開** にチェック → 次のサイト反映（1時間以内）で掲載されます

掲載をやめたいときは **公開のチェックを外す**だけです。

## 表示について

- 声が0件のあいだは「準備中」の案内と講座への導線が表示されます（レイアウトは崩れません）
- ページ下部に「講座を見る／申し込む」のCTAがあり、申込ページへ繋がります
